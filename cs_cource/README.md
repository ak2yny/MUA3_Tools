# MUA3 Tools C# Source

This folder is for source code from other modders.

- [VGAudio](https://github.com/Thealexbarney/VGAudio) is under the MIT Licence and I haven't checked with by [Alex Barney](https://github.com/Thealexbarney) for using the source in this project (which probably doesn't make sense anyway). So, there is only the built assets at this time (Note: no release, built from source).
- [Cethleann by yretenai](https://github.com/yretenai/Cethleann) (MIT) is mainly a format parser and has limited support for actual game files (it supports files from one non-MUA3 game only). Development has stopped.
- [Fatal Frame All-In-One by Jupisoft](https://gamebanana.com/members/2642157) is a combination of many tools by other developers listed, but it includes some repacking tools that don't seem to be available elsewhere. Source is incuded in release and GitHub has been hidden or deleted (Development has stopped).
